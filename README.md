Welcome to my personal developer portfolio! This project serves as a centralized hub for my skills, experience, and development trajectory. Built from the ground up with modern tools, it focuses on performance, accessibility, and high-end design aesthetics.

## ✨ Features

- 🌍 **Full Internationalization (i18n):** Native, dynamic support for Spanish, English, and Russian using custom localized JSON configurations.
- 🎨 **Premium UI/UX:** Built with Tailwind CSS, utilizing modern aesthetics like **Glassmorphism**, dynamic glowing auras, micro-animations, and smooth scroll reveals.
- 📱 **Mobile First:** 100% responsive design with carefully crafted mobile components like an animated hamburger menu and dynamic text sizing.
- ⚡ **Blazing Fast Performance:** Powered by Astro, shipping zero unused JavaScript to the client.
- 📄 **Smart CV Delivery:** The site automatically serves the correct localized version of my CV depending on the user's active language selection.

## 🛠️ Tech Stack

- **Framework:** [Astro](https://astro.build/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Logic:** Vanilla JavaScript (ES6+)
- **Icons:** Custom Inline SVGs & Devicon

## 🚀 Running Locally

Want to check out the code and run it locally? Follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/al3-rom/Portfolio2.0.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd Portfolio2.0
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:4321` to see the magic happen!

## 📂 Architecture & Structure

```text
/
├── public/
│   ├── i18n/           # Translation dictionaries (es.json, en.json, ru.json)
│   └── *.pdf           # Localized Curriculum Vitae files
├── src/
│   ├── assets/         # Reusable SVG Icons and assets
│   ├── layouts/        # Core page layouts and boilerplate
│   └── sections/       # Modular UI sections (Nav, Hero, AboutMe, Exp, etc.)
└── astro.config.mjs    # Global Astro Configuration
```
